CampusConnect RCPIT
===================
CampusConnect RCPIT is a full-stack, web-based learning ecosystem developed for students and faculty of
R. C. Patel Institute of Technology (RCPIT). The platform provides a centralized digital portal for managing
and accessing academic resources such as video lectures, notes, mind maps, and Daily Practice Problem (DPP)
quizzes. Students can track their learning progress, attempt quizzes, and stay updated with institute-level
academic activities.


------------------------------------------------------------
1. Objective / Purpose
------------------------------------------------------------
The main objective of CampusConnect is to provide a unified virtual academic environment that:
- Enhances student learning engagement digitally.
- Simplifies resource sharing for faculty members.
- Offers real-time performance tracking through student dashboards.
- Replaces traditional manual resource distribution systems.

It bridges the gap between students and faculty and enables accessibility from anywhere at any time.


------------------------------------------------------------
2. Key Features
------------------------------------------------------------
- Secure student & faculty login system
- Upload and access study materials (PDF/Notes/PPTs/Recorded Lectures)
- Daily Practice Problems (DPP) with auto evaluation
- Interactive quizzes and results
- Topic-wise mind maps and learning resources
- Notification & updates system
- Student performance analytics dashboard
- Admin panel to manage users and content
- Responsive UI for mobile & desktop

6. Installation / Setup Instructions
------------------------------------------------------------
1. Clone the repository:
   git clone https://github.com/your-username/CampusConnect-RCPIT.git

2. Move to the project folder:
   cd CampusConnect-RCPIT

3. Install backend dependencies:
   npm install

4. Install frontend dependencies (if separate):
   cd client
   npm install

5. Create a .env file containing:
   DATABASE_URL=
   JWT_SECRET=
   PORT=

6. Run the development server:
   npm start
   or
   npm run dev (for frontend frameworks)

7. Open browser and visit:
   http://localhost:3000
